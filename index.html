One-time setup · Takes about 10 minutes
Step 1 of 4
Register an Azure App (free)
This gives the tool secure read/write access to your OneDrive Excel files using your Microsoft 365 login.
  1. Go to portal.azure.com and sign in with your M365 account
  2. Search "App registrations" → click New registration
  3. Name: Telluno BD Cadence
  4. Supported account types: Accounts in any organizational directory and personal Microsoft accounts
  5. Redirect URI: Single-page application → enter the URL of this page (your GitHub Pages URL)
  6. Click Register
  7. On the Overview page, copy the Application (client) ID
  8. Go to API permissions → Add permission → Microsoft Graph → Delegated → select Files.ReadWrite and User.Read → Add
Step 2 of 4
Add a TODAY column to your pipeline files
Open VES_Pipeline.xlsx and Gyre_Pipeline.xlsx in Excel. Add a new column header called TODAY (column M is recommended, after Week Updated).

Each evening or morning, type Y in the TODAY column for every contact you want on that day's cadence list. The tool filters on this — it won't show anyone without a Y.

After logging an outcome, the tool automatically changes that cell to Done so your list self-manages.
Step 3 of 4
Confirm your OneDrive file paths
Enter the paths to your files relative to your OneDrive root folder. Use forward slashes.
Step 4 of 4
Host this file on GitHub Pages
This tool needs to be served from a URL (not opened as a local file) for Microsoft login to work. GitHub Pages is free and takes 5 minutes.
  1. Create a free account at github.com
  2. New repository → name: bd-cadence → set to Public
  3. Upload this HTML file as index.html
  4. Settings → Pages → Source: main branch → Save
  5. Your URL will be: https://[username].github.io/bd-cadence/
  6. Go back to your Azure App Registration → Authentication → update the Redirect URI to this URL
Once hosted, open that URL on your iPhone, add to Home Screen via Share → Add to Home Screen.
